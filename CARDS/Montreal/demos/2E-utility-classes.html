Utility Classes

This library includes some simply utility classes. These classes allow you to apply simple CSS properties on individual elements.

<h3 class="c4-text-align-center">Sample Title</h3>

c4-text-align-left

  • Type: text-align

Align text element to the left.

c4-text-align-center

  • Type: text-align

Align text element to center.

c4-text-align-right

  • Type: text-align

Align text element to the left.

c4-color-primary

  • Type: color

Apply primary color to element.

c4-color-secondary

  • Type: color

Apply secondary color to element.

c4-shadow

  • Type: box-shadow

Apply box-shadow to element.

c4-inline-svg

Sizes an SVG inline with surrounding elements.

c4-ml-auto

  • Type: margin-left

Set margin-left of element to auto. Usefull for pushing element to the right inside a flex container (eg footer).

c4-mr-auto

  • Type: margin-right

Set margin-right of element to auto. Usefull for pushing element to the left inside a flex container (eg footer).
