The Markup

The following details the markup used to create a instance of a Montreal news card along with some of the additional elements that are supported.

A basic card can be created with the following HTML markup…

<div class="c4-montreal">
  <div class="c4-montreal-image">
    <img src="https://source.unsplash.com/FaPxZ88yZrw/400x300" alt="Sample Image">
  </div>
  <div class="c4-montreal-content">
    <p>Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Donec odio. Quisque volutpat mattis eros.</p>   
  </div>
</div>

Hover icon

To add an icon on hover of the card, add your icon code (eg. SVG) just before the img tag wrapped in the following..

<div class="c4-montreal-hover-icon">
  // Your icon code
</div>

Date block

Use the following markup to create a date block. Date blocks use the primary color for the top half and the secondary color for the bottom.

<div class="c4-montreal-date">
  <div class="c4-montreal-date-top">
    10
  </div>
  <div class="c4-montreal-date-bottom">
    Jan
  </div>
</div>


If placed within the .c4-montreal-image div, the date block will display over the image in the opposite corner to the ‘curve’ detail.

<div class="c4-montreal">
  <div class="c4-montreal-image">
    <div class="c4-montreal-date">
      <div class="c4-montreal-date-top">

Meta Info

  <li>
    // Meta item
  </li>
  <li>
    // Meta item
  </li>
  <li>
    // Meta item
  </li>
</ul>

Card footer

A card footer is created with the following markup using the c4-montreal-footer class. This div is added as a first child of the outside .c4-montreal div.

<div class="c4-montreal-footer">
  <p>Lorem ipsum dolor</p>
</div>

Linking a card

To link your card use the following markup as the last child of the outside wrapper of the card.

<a href="#" class="c4-montreal-link"></a>
