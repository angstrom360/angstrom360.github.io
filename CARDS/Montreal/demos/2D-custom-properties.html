Defining Custom Properties

Key variables of the CSS library can be set using https://developer.mozilla.org/en-US/docs/Web/CSS/â€“* either globally or inline within a single or selected number of elements.

Defining custom properties globally

The following example shows how custom properties can be defined globally across your project. Ensure that such CSS is loaded after the library CSS.

.c4-montreal {
  --primary-color: #F6CD14;
  --secondary-color: #D0206E;
}

--text-color

  • Type: Color
  • Default: #555555

Defines the text color within the figcaption area.

--primary-color

  • Type: Color
  • Default: #00B4DB

Defines the primary color.

--secondary-color

  • Type: Color
  • Default: #0083B0

Defines the secondary color.

--style-radius

  • Type: Length
  • Default: 35%

Defines the radius to the image style curve.

--padding

  • Type: Length
  • Default: 25px

Defines the padding of the text area (c4-montreal-content).

--image-row-width

  • Type: Length
  • Default: 50%

Defines the width of the image when using c4-montreal-layout-row.

--border-radius

  • Type: Length
  • Default: 3px

Defines the outside border radius of the card.

--transition-duration

  • Type: Time
  • Default: 600ms

Defines the duration of the animation on hover.
