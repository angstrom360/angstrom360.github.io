The animated button & menu jQuery plugin allows you to easily replace <a> links
with animated buttons and grouping buttons into a menu.

Features:
  1. Buttons of any size.
  2. Any background color, hover color and font.
  3. Allows you to create animated buttons with images.
  4. Create buttons with rounded corners (not supported in IE 7 and IE 8).
  5. Transition easing for animation.
  6. The button can be "selected" and "toggled".
  7. 16 different button animation effects.
  8. You can determine the size of the animated elements.
  9. Button autosize.
  10. Fully customizable - unlimited number of button's appearence combinations.
  11. Buttons can be used in forms.
  12. Buttons can be combined into a menu.
  13. Package contains a plugin for building custom menus.
  14. Package contains a code for drop down menu.

Example - From with Two Color Button

				
The example shows the usage of the lhpTwoColorButton plugin in forms. The 'source code' section shows an easy way to replace the default 'submit' form button with an animated button.

				

					

						
						
						
					

				

				show source code


				
var sampleFormStt = {
	'effect' : 'fx4',
	'particleWidth' : '20',
	'particleHeight' : '20',
	'labelActiveColor' : '#111111',
	'labelInactiveColor' : '#666666',
	'particleColorOne' : '#666666',
	'particleColorTwo' : '#d8d8d8',
	'background-color' : '#f4f4f4',
	'border-radius' : '10',
	'width' : '160',
	'height' : '40',
	'selected' : false,
	'toggle' : false
};
$('#fourthSample input[type="submit"]').each(function () {
	var $this = $(this);
	$this.css('display', 'none');
	$('<a href="#"></a>').text($this.attr('value')).insertBefore($this).lhpTwoColorButton(sampleFormStt)
	.click(function(event) {
		event.preventDefault();
		$this.click();
	});
});
