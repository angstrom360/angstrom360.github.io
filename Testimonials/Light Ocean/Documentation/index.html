Features

  • Pure CSS3 + HTML5 (no JavaScript and jQuery)
  • 20 - Profile card Grid is a Multipurpose Profile Showcase HTML5 Widget
  • CSS3 transitions (hover effects)
  • Responsive layout (resizes automatically on smaller screens)
  • Documentation file included
  • Compatible with Bootstrap.
  • Easy to use and customize.
  • Google Fonts
  • Suitable for all Profile card.
  • Well commented code
  • Font Awesome Icons included
  • Flexible and Responsive Layout: more than one image per line
  • Original Design
