Documentation of

Testimonial-kit
Minimal testimonial showcase

Build with CSS3 & Bootstrap 4.x grid




Initialize the bootstrap-grid.min.css [If you already using bootstrap.css or bootstrap.min.css then you do not need to use it.]


<link href="css/bootstrap-grid.min.css" rel="stylesheet" type="text/css">

Initialize the fontawesome.min.css [If you already using fontawesome icon family then you do not need to use it.]


<link href="css/fontawesome.min.css" rel="stylesheet" type="text/css">

Initialize the testimonial-kit.css


<link href="css/testimonial-kit.css" rel="stylesheet" type="text/css">
